# Programação Orientada a Objetos — Java

Este repositório reúne **todos os códigos em Java utilizados na disciplina de
Programação Orientada a Objetos (POO)**.

Aqui estão disponíveis **exemplos apresentados na apostila**, **códigos de listas
de exercícios** e **projetos de referência**, de forma organizada e acessível
aos alunos.

---

##  Objetivo

Disponibilizar, em um único local, todo o código-fonte em Java utilizado ao
longo da disciplina, permitindo:

- Consulta e estudo dos exemplos apresentados em sala e na apostila  
- Apoio à resolução das listas de exercícios  
- Referência para projetos e atividades avaliativas  

Não há separação por aulas.  
**Todo código citado no material didático está presente neste repositório.**

---

Cada diretório contém apenas **códigos em Java**, mantendo a organização clara
e objetiva.

---

##  Linguagem Utilizada

- Java

Os códigos seguem uma abordagem **didática**, priorizando clareza e compreensão
dos conceitos de Programação Orientada a Objetos, como:

- Classes e objetos  
- Encapsulamento  
- Herança  
- Polimorfismo  
- Abstração  

---

##  Observações Importantes

- Os códigos foram desenvolvidos **exclusivamente para fins educacionais**  
- Alguns exemplos podem ser simplificados propositalmente para facilitar o
  entendimento dos conceitos  
- Recomenda-se a leitura conjunta da apostila da disciplina
